# Case Study: 打开微信

**日期**: 2025-01-24  
**任务**: 使用novaic打开微信并发送消息

---

## 操作流程

### Step 1: 截取桌面
- 调用 `screenshot()` 查看Ubuntu桌面
- 看到Applications菜单、桌面图标

### Step 2: 点击Applications菜单（无zoom）❌
- 估算坐标 (60, 44)
- 直接 `mouse(click, 60, 44)`
- **结果: 失败，菜单没有打开**

### Step 3: 点击Applications菜单（有zoom）✅
- `screenshot(center={70,44}, zoom_factor=3)` 放大查看
- 调整坐标到 (95, 17)
- `mouse(click, 95, 17)`
- **结果: 成功，菜单打开**

### Step 4: 浏览菜单找微信
- 发现Internet子菜单没有微信
- 通过命令行查找：`which wechat` → `/usr/bin/wechat`

### Step 5: 启动微信
- `run_command(command="wechat", background=true)`
- 微信启动，显示登录界面

### Step 6: 点击Log In按钮（无zoom）❌
- 估算坐标 (506, 478)
- `mouse(click, 506, 478)`
- **结果: 失败，没有反应**

### Step 7: 点击Log In按钮（有zoom）✅
- `screenshot(center={506,455}, zoom_factor=2)` 放大查看
- 看到准星偏了，实际按钮在 (680, 478)
- `mouse(click, 680, 478)`
- **结果: 成功，登录进入微信**

### Step 8: 发送操作记录
- 点击File Transfer
- 输入操作记录
- 发送成功

---

## 关键观察

### 失败的点击（无准星校准）
| 目标 | 估算坐标 | 结果 |
|------|----------|------|
| Applications | (60, 44) | ❌ 失败 |
| Log In | (506, 478) | ❌ 失败 |

### 成功的点击（有准星校准）
| 目标 | 估算坐标 | 校准后坐标 | 结果 |
|------|----------|------------|------|
| Applications | (70, 44) | (95, 17) | ✅ 成功 |
| Log In | (506, 455) | (680, 478) | ✅ 成功 |

### 坐标误差分析
- Applications: 误差 ~35px
- Log In: 误差 ~175px（差得很远！）

---

## Token消耗分析

### 如果不用准星校准（假设需要3次重试）

```
Applications菜单:
  screenshot() → 推理 → click(失败) →
  screenshot() → 分析失败 → click(失败) →
  screenshot() → 再分析 → click(成功)
  = 3次完整推理循环

Log In按钮:
  screenshot() → 推理 → click(失败) →
  screenshot() → 分析失败 → click(失败) →
  screenshot() → 再分析 → click(成功)
  = 3次完整推理循环

总计: 6次完整LLM推理循环
```

### 使用准星校准

```
Applications菜单:
  screenshot() → 估算 →
  screenshot(zoom) → 看准星，调整 →
  click(成功)
  = 2次截图 + 简单确认

Log In按钮:
  screenshot() → 估算 →
  screenshot(zoom) → 看准星，调整 →
  click(成功)
  = 2次截图 + 简单确认

总计: 4次截图 + 简单确认逻辑
```

### 对比

| 指标 | 无校准 | 有校准 |
|------|--------|--------|
| LLM完整推理 | 6次 | 2次 |
| 失败点击 | 4次 | 0次 |
| 状态污染风险 | 高 | 无 |

## 结论

1. **准星校准有效** - 2个失败点击都通过校准后成功
2. **误差可能很大** - Log In按钮误差达175px
3. **校准成本低** - 只需额外一次zoom截图
4. **省token** - 避免了失败后的完整推理循环
