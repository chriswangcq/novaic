# 微信操作指南

Linux 桌面版微信的操作技巧和注意事项。

## 基本信息

| 属性 | 值 |
|------|-----|
| 应用名称 | `wechat`（小写） |
| 启动命令 | `wechat` |
| 窗口标题 | 通常包含 "微信" 或 "WeChat" |

## 核心操作流程

### 1. 启动微信

```python
run_command(command="wechat", background=True)
# 等待窗口出现
run_command(command="sleep 3")
list_windows()  # 确认微信窗口已打开
```

### 2. 进入聊天

点击左侧联系人头像进入聊天页面：
- **首次点击头像进入时，输入框默认激活**（光标已在输入框中）
- 无需额外点击输入框，可直接输入

```python
# 点击联系人头像后，直接输入即可
keyboard(action='type', text='你好')
```

### 3. 定位输入框

如果输入框失去焦点，需要重新点击：
- **输入框位置**：在工具栏（表情、文件、截图按钮）的**下方**
- 工具栏是一排小图标，输入框是下面的文本区域

```
工具栏：[😊] [📁] [✂️] [📞] ...   ← 不要点这里
─────────────────────────────────
输入框：|                        ← 点击这里
        |
```

### 4. 输入包含换行的文字 ⚠️

微信输入框中 Enter 键会直接发送消息，**无法通过键盘输入换行**。

**必须使用剪贴板**：

```python
# 1. 将文字写入剪贴板
set_clipboard(content="""第一行
第二行
第三行""")

# 2. 粘贴到输入框
keyboard(action='key', keys=['ctrl', 'v'])

# 3. 发送
keyboard(action='key', keys=['Return'])
```

### 5. 发送消息

```python
# 方式一：按 Enter 发送
keyboard(action='key', keys=['Return'])

# 方式二：点击发送按钮（如果有）
# 发送按钮通常在输入框右侧或下方
```

## 常见场景

### 场景 A：给某人发送简单消息

```python
# 1. 截图找到联系人
screenshot()

# 2. 点击联系人头像（进入后输入框自动激活）
mouse(action='aim', x=头像X, y=头像Y)
mouse(action='click', aim_id='...')

# 3. 直接输入（无需点击输入框）
keyboard(action='type', text='你好，在吗？')

# 4. 发送
keyboard(action='key', keys=['Return'])
```

### 场景 B：发送多行文字

```python
# 1. 进入聊天（同上）

# 2. 使用剪贴板输入多行文字
set_clipboard(content="""项目进度汇报：
1. 功能开发完成 ✓
2. 测试进行中
3. 预计明天上线""")

keyboard(action='key', keys=['ctrl', 'v'])

# 3. 发送
keyboard(action='key', keys=['Return'])
```

### 场景 C：搜索联系人

```python
# 点击搜索框（通常在联系人列表上方）
# 或使用快捷键 Ctrl+F
keyboard(action='key', keys=['ctrl', 'f'])
keyboard(action='type', text='张三')
```

## ⚠️ 注意事项

1. **输入框位置**：在工具栏**下方**，不要误点工具栏图标
2. **换行必须用剪贴板**：Enter 会直接发送，不能用于换行
3. **首次进入自动激活**：点击头像进入聊天后，输入框默认有焦点
4. **中文输入**：确保系统输入法正确，`type` 支持直接输入中文
5. **等待加载**：切换聊天后稍等片刻，确保消息历史加载完成
