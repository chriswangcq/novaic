# Desktop CI/QA Handoff - Round 006

## Goal
Make desktop release evidence package executable by non-authors and reusable by CI/QA.

## Required Commands
1. `cargo check --manifest-path "novaic-app/src-tauri/Cargo.toml"`
2. `npm run tauri:build`
3. `novaic-app/scripts/validate_fresh_profile.sh`

## Required Evidence Files
- `ops-rounds/round-006/20-reports/desktop-evidence/fresh-profile-*-summary.txt`
- `ops-rounds/round-006/20-reports/desktop-evidence/fresh-profile-*-startup.log`
- `ops-rounds/round-006/20-reports/desktop-evidence/fresh-profile-*-startup-diagnostics.jsonl`
- `ops-rounds/round-006/20-reports/desktop-evidence/fresh-profile-*-metadata.txt`
- `ops-rounds/round-006/20-reports/team-desktop-report.md`

## Failure-Mode Examples (for triage drills)

### Example A: Missing app bundle path
- trigger command:
  - `novaic-app/scripts/validate_fresh_profile.sh "/tmp/not-exists/NovAIC.app"`
- expected result:
  - exit code non-zero
  - error starts with `ERROR: app bundle not found`
- operator action:
  - re-run `npm run tauri:build`
  - verify artifact path exists before replay

### Example B: Startup diagnostics contains startup-blocking status
- trigger condition:
  - dependency startup failure or port conflict that survives preflight cleanup
- expected result:
  - `fresh-profile-*-summary.txt` reports `error_timeout_count>0`
  - script exits non-zero
- operator action:
  - inspect `fresh-profile-*-startup-diagnostics.jsonl`
  - classify via `ops-rounds/round-002/20-reports/desktop-startup-triage-matrix.md`
  - only mark pass after rerun reaches `error_timeout_count=0`

## QA/CI Pass Criteria
- At least one replay run has:
  - `event_count>=10`
  - `error_timeout_count=0`
  - stage chain includes `tools-server`
- Evidence bundle archive and manifest are generated by:
  - `novaic-app/scripts/build_desktop_evidence_bundle.sh`
