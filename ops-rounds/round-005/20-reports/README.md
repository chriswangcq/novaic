# Round 005 Reports (Decision-Driven)

Each team must submit one report by 18:00 with:
- Completed work
- Command evidence + pass summary
- Artifacts/docs paths
- Acceptance mapping
- Risks / blockers
- **Decision Needed (required when any risk/blocker exists)**
  - issue
  - options (A/B/C)
  - recommendation
  - impact
- Self status

## Hard Rule
- If a team has blocker/risk but omits `Decision Needed`, report is considered incomplete.
