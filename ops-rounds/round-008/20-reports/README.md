# Round 008 Reports (Execution Mandatory)

Each team must submit one report with:
- implemented work (not interpretation)
- exact commands + pass/fail summary
- artifact/doc paths
- acceptance mapping
- risks/blockers
- Decision Needed (if any): issue/options/recommendation/impact/owner/deadline
- self status

## Hard Rule
- Understanding-only updates are invalid.
- No executed commands + no artifacts => cannot mark `DONE`.
