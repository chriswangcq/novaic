refactor: macOS path fixes, frontend chat improvements, backend enum standardization, and v24 runtime summary

Major refactor with cross-platform improvements, frontend performance enhancements, and new runtime summary features.

## macOS Cross-Platform Path Fixes
- Fixed 23 hardcoded paths across Python/Rust/Shell scripts
- Implemented cross-platform data directory detection (macOS/Linux/Windows)
- Updated VM manager, SSH, setup scripts to use tempfile.gettempdir()
- Fixed QEMU socket paths to use platform-agnostic temporary directories

## Frontend Chat Component Refactor
- Removed StreamingText, ThinkingBlock, ToolCallCard components
- MessageList: Added virtual list with useVirtualList hook for performance
- Implemented scroll pagination with useScrollPagination hook
- Added unread message count tracking
- Improved scroll-to-bottom behavior with ref-based API
- New hooks: useVirtualList, useScrollPagination, useAutoScroll
- New constants: constants/scroll.ts

## Backend Enum Standardization
- New module: common/enums.py with RuntimeStatus, SubagentStatus, RuntimePhase, etc.
- Replaced hardcoded status strings with enum values throughout codebase
- Improved type safety and consistency

## Runtime Summary v24 Features

- Internal API: New endpoints for HRL operations (/hrl, /hrl/add, /hrl/remove)
- Summary handlers: New handlers for hot/cold summary generation

## Configuration Management
