# Roles and Responsibilities

## Program Owner
- opens each round
- approves gates and final decision
- resolves cross-team priority conflicts

## Team Owner
- executes assigned dispatch tasks
- submits evidence-backed reports before round close
- raises decision-needed items with owner and target_round

## Round Feedback Owner (Program Owner)
- validates evidence completeness and correctness
- records round feedback by severity
- issues PASS / CONDITIONAL_PASS / FAIL recommendation

## Platform/Governance Owner
- maintains governance rules and templates
- keeps CI checks aligned with governance docs
- ensures no policy drift across rounds
