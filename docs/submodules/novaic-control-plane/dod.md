# Definition of Done

A task is done only if all conditions are met:

1. Implementation exists.
2. Acceptance criteria are satisfied.
3. Evidence is attached:
   - command(s)
   - pass/fail summary
   - artifact/doc path
4. No unresolved blocker in the task scope.

## Invalid completion examples

- "Analyzed and understood the issue"
- "Will implement next round"
- "Tested mentally"
