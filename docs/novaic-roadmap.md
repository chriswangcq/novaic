  - action: wait

## 📝 决策记录

### 2026-01-23: 定位转变

**背景**：从通用 MCP Server 转向 "AI Computer"

**决策**：
1. 重命名 linux2mcp → NovAIC
2. 强调持久化、记忆、环境感知
3. 与 E2B 等临时沙箱差异化
4. 创造新品类：AIC (AI Computer)

**原因**：
- 持久化是独特价值
- 市场空白
- 更清晰的产品定位
- NovAIC 品牌独特，避免与 OpenStack Nova 冲突

### 2026-01-23: 技术选型

**背景**：Docker vs QEMU

**决策**：选择 QEMU

**原因**：
- 完整 VM 隔离
- 真正的持久化（QCOW2）
- 支持快照和回滚
- 更接近"真电脑"体验
