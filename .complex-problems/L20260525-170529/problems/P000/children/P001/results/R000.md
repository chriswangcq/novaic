# Backend public_title projection implemented

## Summary

Added explicit `public_title` support to the backend activity projection path so Agent Monitor row titles can be structured public data rather than frontend reasoning-text inference.

## Done

- Added `public_title` to `novaic-common` app-facing Entangled contract for `agent-activity-records`.
- Added `public_title` to `novaic-business` schema push definition for `agent_activity_records`.
- Updated `novaic-agent-runtime` activity projection to materialize `public_title` on every emitted record.
- Added runtime mapping for shell `agentctl im read` -> `读取了你的消息` and shell `agentctl im reply` -> `已回复你`.
- Preserved reasoning `text` content while setting reasoning `public_title` to structured `完成思考`.
- Updated common/runtime tests to assert `public_title` exists and carries expected values.

## Verification

- `python3 -m pytest tests/test_pr193_activity_projection.py` in `novaic-agent-runtime`: 8 passed.
- `python3 -m pytest tests/test_pr166c_entangled_app_entities.py` in `novaic-common`: 2 passed.
