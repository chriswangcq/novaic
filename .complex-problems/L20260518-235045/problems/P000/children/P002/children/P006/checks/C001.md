# 后端核心服务代码调研成功验证

## Summary

三个核心后端服务的代码调研覆盖了所有成功标准要求的维度，事实均来自源代码直接阅读。

## Evidence

- agent-runtime 事实清单覆盖：入口文件 (main_novaic.py + 9 子命令)、6 种 worker 类型及生产 roster、5 种 saga 定义含完整步骤流程、queue_service 完整目录树、45+ API 路由

- 服务间调用关系以表格形式记录：调用方→被调用方→协议→端点→用途
