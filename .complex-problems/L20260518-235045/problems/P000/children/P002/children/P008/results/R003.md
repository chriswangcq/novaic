# 前端与桌面端代码调研结果

## Summary

通过两组并行代码调研，完成了对 novaic-app（Tauri 2.x + React 18 客户端）、VmControl（Rust 嵌入式设备控制 crate）和 novaic-mcp-vmuse（Python VM 内桌面控制工具）的全面代码级调研。React 组件结构、Tauri 命令体系、状态管理、路由、Entangled 客户端集成、VmControl WebRTC/Scrcpy 集成、mcp-vmuse 工具注册和桌面控制能力、前后端通信方式均已确认。

## Done

- **novaic-app React 组件结构**：按功能分目录，src/components/ 下 13 个子目录（Agent/Auth/Chat/Console/Layout/Settings/VM/Visual 等），入口链路 main.tsx → App.tsx → QueryClientProvider + EntangledProvider + AppErrorBoundary → AuthPage | LayoutContainer
- **Tauri 命令体系**：27 个注册命令（src-tauri/src/commands/ + entangled_client crate），涵盖 Gateway 代理（gateway_get/post）、安全存储（secure_storage_*）、文件缓存（get_cached_file/fetch_cached_bytes）、音频录制、WebRTC 信令、Entangled 同步（entangled_method 唯一写入入口）
- **状态管理**：Zustand（主 store + 设备状态 store）+ TanStack Query（staleTime=Infinity，仅由 Tauri 事件 entities_changed 触发 invalidation），服务层单例按 userId 作用域（SyncService/AgentService/ModelService 等 8 个）
- **路由**：不使用 react-router，通过自定义布局状态（primaryTab + narrowPage）实现，桌面三栏（PrimaryNav | AgentDrawer | Main）+ 移动单栏 + BottomTabBar
- **Entangled 客户端集成**：Rust crate（Entangled/packages/client-rust/），WS 协议（Entangle/Disentangle/Action/Sync/Ack/Schema），SQLite 持久化缓存（per-user），悲观写入 + 乐观展示架构，15 个同步实体（users/agents/devices/messages/skills 等），React 侧通过 src/data/entangled/ 绑定（createListStore/createFormStore/createStreamStore）
- **VmControl**：独立 Rust workspace crate（src-tauri/vmcontrol/），嵌入式服务（start_embedded_server，无 HTTP 端口），axum Router + CloudBridge Router::oneshot() 零网络开销分发；WebRTC 统一入口按 DeviceKind（LinuxVm/Android/HostDesktop）路由；VideoBroadcaster 一对多广播 + 全局 UDPMux + Trickle ICE；Scrcpy 3.3.4 tunnel_forward 协议；Cloud Bridge 30 种类型化命令通过 WS 连接 Device Service；50+ API 路由
- **novaic-mcp-vmuse**：Python aiohttp HTTP JSON 服务器（:8080），运行在 VM 内部，6 类工具：Desktop（截图/鼠标 aim-execute 两阶段/键盘，xdotool）、Browser（Playwright 9 个端点）、Shell（命令执行 + Python）、File（读写 + 二进制）、Window（wmctrl 7 个端点）、Context（系统/目录/应用/剪贴板/环境 7 个端点），VmControl 通过 VMUSE 代理路由转发请求
- **前后端通信**：前端零直连——所有通信通过 Tauri IPC → Rust 进程；三通道：GatewayClient HTTP 代理（auth/api/blob）、AppBridge WS（信令 + push 事件 + Entangled endpoint 发现）、EntangledSyncBridge WS（实体数据同步 + 写入）；6 个 Tauri 事件（entities_changed/schema_changed/auth_rejected/gateway_push/bridge_connected/disconnected）
