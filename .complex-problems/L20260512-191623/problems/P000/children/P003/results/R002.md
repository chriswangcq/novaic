# Cortex empty display projection diagnostic added

## Summary

Changed Cortex display perception formatting so an empty display projection emits a diagnostic text marker instead of ambiguous `OK`, while preserving image projection behavior.

## Done

- Added `empty_text` parameter to `_format_mcp_content`.
- `format_for_display_perception_llm` now uses `[Display produced no text or media for LLM perception]` when no text/media is available.
- Added a unit test for empty display projection.
