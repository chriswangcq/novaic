# Ticket: Reconcile ContextEvent child outcomes

## Goal

Build a compact reconciliation table for P421-P424, including child result IDs, check IDs, test evidence, source changes, and residual risks.

## Acceptance Criteria

- P421, P422, P423, and P424 are all accounted for.
