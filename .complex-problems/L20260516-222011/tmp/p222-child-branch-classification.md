# Classify active projection branch sites

## Problem

Remaining projection code contains branch handling for shell output, display perception, MCP content, image data URLs, and unknown dicts. Confirm each branch is intentional and not an old inactive path.

## Success Criteria

- Active projection branch sites are enumerated with file/line evidence.
- Each branch is classified as intentional or follow-up-worthy.
- Any branch that is stale, ambiguous, or untested creates follow-up work.
