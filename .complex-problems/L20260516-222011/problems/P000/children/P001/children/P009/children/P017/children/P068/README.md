# Active docs migration and legacy residue scan

## Problem

Active docs may still describe migration/fallback/compatibility behavior as current guidance even after the final architecture has moved on.

## Success Criteria

- Focused scans cover active docs for TODO, FIXME, legacy, compat, fallback, migration, and old-path wording.
- Historical roadmap or ticket material is classified separately from active guidance.
- Small active-doc wording cleanup is applied directly if safe.
- Larger architecture/documentation ambiguity is routed to explicit child/follow-up work.
