# Active docs and scripts semantic residue discovery

## Problem

Audit active docs and scripts for stale service-boundary language that could mislead future agents or operators.

## Success Criteria

- Focused scans cover docs, root scripts, deployment scripts, and operational notes.
- Findings classify active stale claims versus historical/roadmap notes.
- Exact doc/script remediation candidates are listed.
- No product files are modified in this discovery child.
