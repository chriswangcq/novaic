# Classify llm-factory as intentionally standalone

## Proposed Solution

## Acceptance Criteria

- llm-factory is classified with rationale.
- No stale references to llm-factory or port 9100 exist in launch scripts.
- If the classification is "intentionally standalone", no code changes needed.

## Verification Plan

- Read llm-factory README for its stated purpose.
