# Shell artifact manifest output contract discovery result

## Summary

Completed the shell artifact manifest contract discovery. The active shell behavior is good: `devicectl hd screenshot` decodes the device base64 response internally, uploads bytes to Blob as `runtime-artifact`, prints a bounded `tool-output.v1` manifest with `blob://runtime-artifact/...`, and excludes raw screenshot/base64 from stdout. The runtime shell output contract also bounds media-like stdout as terminal text and keeps raw stdout only in durable payload diagnostics.
