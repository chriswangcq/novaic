# Packaged backend launcher remediation result

## Summary

Remediated stale packaged backend launcher messaging in both the source Tauri resource and generated apple asset copy. The scripts no longer claim to start all backend services, no longer print a fake Agent Runtime port, and now explicitly identify the packaged backend subset.

## Done

- Updated `novaic-app/src-tauri/resources/backends/start-backends.sh` from "Start all backend services" to "Start the packaged backend subset".
- Removed unused `CONFIG_FILE` and stale `AGENT_RUNTIME_PORT=19991` from the source resource launcher.
- Updated source resource startup/final messages to say it starts Blob Service, Gateway, and Agent Runtime process, and does not start Entangled, Business, Device, Cortex, or Sandboxd.
- Applied the same change to `novaic-app/src-tauri/gen/apple/assets/backends/start-backends.sh` to keep the checked-in generated apple asset copy consistent with the source resource.
