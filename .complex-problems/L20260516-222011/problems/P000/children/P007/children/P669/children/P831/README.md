# LLM Factory log API and UI audit

## Problem

LLM Factory provides a log viewing UI (factory-logs.html) and associated API endpoints. Verify the UI file exists, API paths referenced are valid, and the fetch/display contract is consistent.

## Success Criteria

- factory-logs.html and its associated API endpoints located and inspected.
- Fetch paths in UI code match backend route definitions.
- No stale API references or broken fetch paths.
