Classification:

- `novaic_cortex/api.py`: `ScopeEndRequest.remaining_stack` is the explicit diagnostic request field; `_archive_diagnostics(req)` copies it into nested `archive_diagnostics.remaining_stack` only when the request supplied a complete diagnostic contract.

- `novaic_cortex/context_event_writer.py`: top-level `remaining_stack: list[str]` and optional nested `archive_diagnostics` are written as distinct fields.
- `novaic_cortex/context_event_projection.py`: projection consumes only top-level `event.payload.get("remaining_stack")`; it does not inspect nested `archive_diagnostics.remaining_stack`.
- Tests assert both the semantic top-level stack and nested diagnostic stack payloads separately.
