# Audit runtime history versus media perception boundary

## Problem

Runtime context assembly must keep ordinary tool history text-only while allowing current display perception to become model-native image content. This belongs under P130 because provider media safety starts before the factory adapter.

## Success Criteria

- Runtime files/functions for display projection and multimodal context assembly are mapped.
- Tests prove historical tool images are not re-injected as media/text.
- Tests prove current display perception can become model-native media content only at the intended boundary.
