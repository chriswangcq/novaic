# Result: Cortex helper namespace migration

## Summary

Migrated Cortex test helpers out of the generic `tests` namespace and updated Cortex tests to import the package-specific `cortex_tests` namespace.

## Done

- Created `novaic-cortex/cortex_tests/__init__.py` with the explicit helper-package purpose.
- Moved `workspace_test_utils.py` and `blob_boundary_policy.py` from `novaic-cortex/tests/` to `novaic-cortex/cortex_tests/`.
- Rewrote all Cortex test imports from `tests.workspace_test_utils` / `tests.blob_boundary_policy` to `cortex_tests.workspace_test_utils` / `cortex_tests.blob_boundary_policy`.
- Deleted empty `novaic-cortex/tests/__init__.py` so Cortex tests no longer claim the top-level `tests` package.

## Verification

- `rg -n '^\s*(from|import)\s+tests(\.|\s|$)' novaic-cortex/tests || true` -> no output.
- `find novaic-cortex/tests -maxdepth 1 -type f -name '__init__.py' -print` -> no output.
- `rg -n '^\s*from cortex_tests\.' novaic-cortex/tests | wc -l` -> `40`.
