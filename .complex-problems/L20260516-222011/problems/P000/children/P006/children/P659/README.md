# Docs Architecture Stale Claims Audit

## Problem

Search architecture/docs/runbooks/roadmap material for stale claims, old tool names, old paths, and misleading migration residue that can mislead future implementation.

## Success Criteria

- Runs focused docs scans for stale contracts and old path/tool terms.
- Classifies high-signal hits as current, historical/archived, or misleading.
- Updates or deletes misleading docs that are active enough to affect current work.
- Records historical material that is intentionally retained.
