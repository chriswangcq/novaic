# Runtime Stale Backing Path Guard Result

## Summary

Audited the runtime guard for copied `novaic-cortex-sandbox-*` backing paths. The guard is in the Cortex shell execution boundary and rejects stale paths before sandboxd/local-executor selection. I added a quoted-path regression test to cover a realistic pasted command shape.

## Done

- Located `_reject_ephemeral_sandbox_paths` in `novaic-cortex/novaic_cortex/sandbox.py`.
- Confirmed the guard scans the full shell command for `novaic-cortex-sandbox-` before execution and returns exit code `-2` with a message pointing to `/cortex/ro`, `/cortex/rw`, `$RO`, and `$RW`.
- Confirmed the guard runs before the `sandbox_executor is None` branch, so it catches stale paths without attempting local fallback or sandbox execution.
- Added `test_shell_rejects_quoted_ephemeral_cortex_backing_paths_before_execution` in `novaic-cortex/tests/test_sandbox_requires_mount_namespace.py`.
