# Stable Path Schema and Help Guidance Result

## Summary

Audited stable Cortex workspace path guidance in the canonical shell schema and generated Cortex help. Guidance was present, but the common schema test suite did not independently guard it, so I added a public-boundary regression test.

## Done

- Confirmed `novaic-common/common/tools/llm_builtin.py` tells agents to use `/cortex/ro`, `/cortex/rw`, `$RO`, and `$RW` and not copy/reuse `novaic-cortex-sandbox-*` backing paths.
- Confirmed generated `cortex --help` says stable `/cortex/ro` and `/cortex/rw` paths are the contract and backing temp paths are not part of the contract.
- Added `test_shell_schema_lists_stable_cortex_workspace_paths` in `novaic-common/tests/test_tool_definitions_contract.py` to guard the canonical LLM-facing shell schema.

## Verification

- `PYTHONPATH=novaic-common:novaic-cortex:novaic-logicalfs:novaic-sandbox-sdk pytest -q novaic-common/tests/test_tool_definitions_contract.py novaic-cortex/tests/test_tool_schemas_limits.py` -> 29 passed.
- Generated `cortex --help` smoke asserted `/cortex/ro`, `/cortex/rw`, and `backing temp paths are not part of the contract` -> passed.
- `rg -n "/cortex/ro|/cortex/rw|\$RO|\$RW|novaic-cortex-sandbox" novaic-common/common/tools/llm_builtin.py novaic-cortex/novaic_cortex/shell_capabilities.py novaic-common/tests/test_tool_definitions_contract.py novaic-cortex/tests/test_tool_schemas_limits.py` confirmed expected schema/help/test hits only.
