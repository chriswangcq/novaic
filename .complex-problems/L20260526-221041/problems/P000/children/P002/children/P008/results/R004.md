# Stable streaming reasoning projection implemented

## Summary

Runtime activity projection now supports stable running/final reasoning row updates and bounded coalescing for streamed reasoning text.

## Done

- Added `project_reasoning_stream_update()` that reuses the existing `assistant:reasoning` seed for stable record IDs.
- Running updates write `status=running`, `title/public_title=正在思考`, and current reasoning text.
- Final updates write completed status and final reasoning text.
- Added `ReasoningStreamProjector` to suppress small intermediate updates and always emit final text.
- Added tests for stable record IDs, running/final state, and coalescing behavior.
