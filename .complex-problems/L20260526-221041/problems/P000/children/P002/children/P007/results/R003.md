# Runtime Factory stream aggregator implemented

## Summary

Runtime Factory client now supports normalized Factory SSE streaming and aggregates it back into the complete OpenAI-style chat completion shape expected by existing saga code.

## Done

- Added Factory SSE event parsing for `data:` lines, `[DONE]`, keepalives, delta/done/error event types.
- Added `FactoryStreamAccumulator` for content, reasoning, tool-call fragments, finish reason, usage, and optional Factory metadata.
- Added `FactoryLLMClient.stream_chat()` with `stream=true` payloads and optional delta callback.
- Added focused tests for SSE parsing, response aggregation, tool-call argument fragment stitching, callback delivery, and stream error events.
