# Factory route streaming wired

## Summary

Factory `/v1/chat/completions` now honors `stream=true` for providers with `stream_chat()`, returning normalized SSE chunks and completing/failing Factory logs consistently. Unsupported streaming providers fail with a clear HTTP 400.

## Done

- Added `StreamingResponse` route branch for streaming chat completions.
- Added normalized SSE JSON envelope with `type=delta`, provider chunk fields, and terminal `type=done`.
- Added streaming log completion for finish reason, tool-call names, usage, latency, and attempt count.
- Added explicit unsupported-provider failure and log marking.
- Added route-level tests for streaming success and unsupported provider failure.

## Verification

- `cd novaic-llm-factory && python -m pytest tests/test_chat_routes.py -q` passed: 17 tests.
