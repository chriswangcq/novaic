# ActivityTimeline same-record streaming update support

## Summary

Added an explicit same-record update fingerprint for `ActivityTimeline` so streaming reasoning mutations on an existing row can trigger follow-bottom behavior without requiring a new row. Preserved the existing Entangled activity path and kept `updated_at` from synced activity records through normalization.
