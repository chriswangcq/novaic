# Shell Contract Result

## Summary

Implemented the shell terminal-preview contract. The LLM-facing `content` now contains only `tool-output.v1` terminal text and small diagnostics. Full raw stdout/stderr is moved to a durable step payload envelope so Cortex RO payload inspection can still replay the complete shell result.

## Done

- Removed `raw_shell_result` from shell `tool-output.v1.diagnostics`.
- Added durable payload envelope `tool-step-payload.v1` with `llm_content` and raw shell result.
- Updated step persistence to store `durable_payload` while building observations from bounded public `content`.
- Updated Cortex step projection to read `llm_content` from `tool-step-payload.v1`.
- Updated shell output contract tests.
